The "Toronto Death Statistics" Shiny application
Created as a final project for the Coursera "Developing Data Products" class

Author: Andrew Parsons
Date: Sep 24, 2015

Overview:

The Toronto Death Statistics app shows patterns of death statistics by region and over time. 

Date Source:

Toronto's Death Registry Statistics via the City Clerk's office. This data is freely available via Toronto's 
Open Data site (http://www1.toronto.ca/wps/portal/contentonly?vgnextoid=1a66e03bb8d1e310VgnVCM10000071d60f89RCRD)

User Interaction:

Users of the app can select the year and region and be provided aggregate statistics on deaths, 
as well as the pattern throughout the year.
